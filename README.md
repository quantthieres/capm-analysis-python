# CAPM Analysis in Python

This project implements the Capital Asset Pricing Model (CAPM) using historical market data.

## Objectives

- Estimate beta for selected assets
- Analyze the relationship between risk and return
- Compare asset performance relative to the market

## Methodology

- Collect historical price data
- Compute returns
- Estimate CAPM using linear regression

## Tools

- Python
- Pandas
- NumPy
- Statsmodels
